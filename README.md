# pkmapp

A local-only personal knowledge management app.

- TypeScript / web
- Markdown
  - WYSIWYG editing
  - Tree-sitter incremental parsing
- File system access API

## Contributing

See [`CONTRIBUTING.md`](CONTRIBUTING.md) for details.

## Building

1. Clone this repo.
1. `git submodule update --init --recursive`
1. `npm install`
1. `npm run build`
1. `npm run dev`

## Testing

1. `npm run test`
